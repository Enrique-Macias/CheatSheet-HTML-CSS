HTML5 Cheat Sheet

HTML5 is the latest version of HTML. It is designed to be easy to use and understand. This Cheat Sheet will teach you the basics of HTML5.

HTML5 Syntax

HTML5 is the fifth and latest major version of HTML, which aims to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers and devices such as web browsers.

Doctype Declaration: HTML5 introduces a simplified doctype declaration: <!DOCTYPE html>.

Semantic Elements: HTML5 includes new semantic elements like <header>, <footer>, <article>, <section>, <nav>, and <aside> to better structure the content.

Multimedia Elements: New elements such as <audio> and <video> provide native support for embedding media.

Form Enhancements: HTML5 introduces new form controls like <input type="email">, <input type="date">, and attributes such as placeholder, required, and pattern for improved user input.

Global Attributes: Attributes like contenteditable, draggable, and hidden can be applied to any HTML element to enhance functionality.

APIs and Integration: HTML5 includes APIs such as the Geolocation API, Web Storage API, and the Canvas API, enabling richer web applications.

Basic HTML Structure


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Section Divisions


<div> - Division
<header> - Header
<nav> - Navigation
<section> - Section
<article> - Article
<aside> - Aside
<footer> - Footer
<span> - Inline Section

Common Tags


<h1> - Heading (h1-h6)
<p> - Paragraph
<strong> - Bold
<em> - Italic
<a href="url"> - Hyperlink
<img src="url" alt="description"> - Image
<ul> - Unordered List
<ol> - Ordered List
<li> - List Item
<br> - Line Break

Tables


<table> - Define a Table
<tr> - Table Row
<th> - Table Header
<td> - Table Data

Forms


<form> - Form input group declaration
<input> - Input field within form
<select> - Select from drop down list

Miscellaneous Tags


<!-- This is a comment --> - Comment
<!DOCTYPE html> - Document Type Declaration
<meta> - Metadata information tag
