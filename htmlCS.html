HTML5 Cheat Sheet

HTML5 is the latest version of HTML. It is designed to be easy to use and understand. This Cheat Sheet will teach you the basics of HTML5.

HTML5 Syntax

HTML5 is the fifth and latest major version of HTML, which aims to improve the language with support for the latest multimedia while keeping it easily readable by humans and consistently understood by computers and devices such as web browsers.

Basic HTML Structure: Learn the fundamental structure of an HTML5 document, including the simplified doctype declaration and essential elements.

Common Tags: Discover frequently used HTML tags such as headings, paragraphs, links, and lists that form the core building blocks of web pages.

Section Divisions: Understand the new semantic elements in HTML5 that help structure your content more meaningfully, such as <header>, <footer>, <article>, and more.

Tables: Learn how to create and style tables to organize data effectively on your web pages.

Lists: Explore the various types of lists available in HTML5, including ordered and unordered lists, and how to use them.

Miscellaneous Tags: Familiarize yourself with other useful tags and attributes that enhance your web pages, such as comments, metadata, and more.

Basic HTML Structure


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Document Outline


<!DOCTYPE> - Document Type Declaration
<html> - Root Element
<meta> - Character Set
<head> - Page Information
<title> - Page Title
<body> - Page Content

Document Structure


<h[1-6]> - Heading (h1-h6)
<p> - Paragraph
<div> - Page Division
<span> - Inline Selection
<br> - Line Break
<hr> - Horizontal Rule

Tables


<table> - Define a Table
<thead> - Table Header
<tbody> - Table Body
<td> - Table Data
<tr> - Table Row
<th> - Table Header
<caption> - Caption
<tfoot> - Table Footer 
<colgroup> - Column Group
<col /> - Column

Form Events


<form> - Form input group declaration
<fieldset> - Collection of Fields
<legend> - Form Legend
<input> - Input field within form
<select> - Select from drop down list
<optgroup> - Group of Options
<option> - Dropdown Option
<textarea> - Large Text Field
<button> - Button

Lists

<ul> - Unordered List
