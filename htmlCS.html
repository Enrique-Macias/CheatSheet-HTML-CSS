HTML5 Cheat Sheet

HTML5 is the latest version of HTML. It is designed to be easy to use and understand. This Cheat Sheet will teach you the basics of HTML5.

HTML5 Syntax

HTML5 is a language for describing web pages. HTML stands for Hyper Text Markup Language. A markup language is a set of markup tags. HTML documents are described by HTML tags. Each HTML tag describes different content.

Basic HTML Structure


<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Common Tags


<h1> - Heading (h1-h6)
<p> - Paragraph
<strong> - Bold
<em> - Italic
<a href="url"> - Hyperlink
<img src="url" alt="description"> - Image
<ul> - Unordered List
<ol> - Ordered List
<li> - List Item
<br> - Line Break

Section Divisions


<div> - Division
<header> - Header
<nav> - Navigation
<section> - Section
<article> - Article
<aside> - Aside
<footer> - Footer
<span> - Inline Section

Tables


<table> - Define a Table
<tr> - Table Row
<th> - Table Header
<td> - Table Data

Miscellaneous Tags


<!-- This is a comment --> - Comment
<!DOCTYPE html> - Document Type Declaration
<meta> - Metadata information tag
